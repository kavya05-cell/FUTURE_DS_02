# Social Media Campaign Performance Tracker

This project analyzes Facebook and Instagram ad campaign data to evaluate marketing performance using key metrics such as CTR, CPC, and ROI. The goal is to help businesses understand which campaigns generate engagement and revenue, and how future campaigns can be optimized.

## Tools Used
- Power BI – for dashboard creation and data visualization  
- Excel / Google Sheets – for data cleaning and metric calculation  
- Kaggle – for dataset  

## Key Metrics
- Impressions  
- Clicks  
- Click Through Rate (CTR)  
- Cost Per Click (CPC)  
- Revenue  
- Return on Investment (ROI)

## Dashboard Features
- Overall campaign KPIs (Impressions, Clicks, CTR, Spend, ROI)  
- Campaign-wise performance comparison  
- Revenue vs Spend analysis  
- Click trends over time  
- Device and platform performance  
- Interactive filters for date, campaign, and device  

## Insights
- Some campaigns generate high engagement but low profitability, indicating poor conversion efficiency.  
- A few campaigns deliver significantly higher ROI, making them ideal candidates for increased budget allocation.  
- Mobile users show higher engagement than desktop users, suggesting future campaigns should be optimized for mobile devices.

## Recommendations
- Increase budget for high-ROI campaigns.  
- Pause or redesign ads with high CPC and low CTR.  
- Optimize landing pages and creatives for mobile users.  

This project demonstrates how marketing data can be transformed into actionable business insights using data analytics and visualization.
